Keyboard & Input Issues

Category: Keyboard & Input

Having trouble with your Chromebook's keyboard or touchpad? Try these simple troubleshooting steps for common input problems.

Keys Not Working

What to Try:

  1. Restart your Chromebook
  2. Test which keys aren't working by typing the alphabet
  3. Turn the Chromebook upside down and gently shake it to remove debris
  4. Use compressed air to blow between the keys (if available)
  5. Check Settings > Device > Keyboard to make sure settings are correct
  6. Try an external USB keyboard to see if it works

Temporary Workaround:

Press Ctrl + Shift + K to open the on-screen keyboard, or go to Settings > Accessibility > Keyboard > On-screen keyboard.

Touchpad Not Responding

What to Try:

  1. Restart your Chromebook
  2. Make sure the touchpad is clean - wipe it with a soft, dry cloth
  3. Check Settings > Device > Touchpad to verify settings
  4. Try tapping the touchpad instead of clicking
  5. Check if an external mouse works (if one is plugged in, unplug it and see if touchpad starts working)

Missing Key Caps

What to Do:

If key caps are missing or broken:

  1. Look for the missing key cap in your backpack or around your workspace
  2. Do not try to reattach it yourself
  3. Submit a support ticket - key cap replacement requires technical service
  4. Use the on-screen keyboard as a temporary solution
Quick Tip: Most keyboard and touchpad issues can be fixed with a simple restart. Try that first before other troubleshooting steps.

Still Not Working?

If you've tried these steps and your issue isn't resolved, please submit a support ticket using our intake form. Include which keys or touchpad functions aren't working and what you've already tried.
