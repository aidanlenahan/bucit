INVENTORY FEATURE - FILE STRUCTURE
=====================================

New Files Created:
------------------
inventory.php                           → Main inventory management page
phpscripts/create_inventory_tables.sql  → Database schema setup
INVENTORY_SETUP.md                      → Setup & usage documentation

Modified Files:
---------------
edit_ticket.php                         → Added parts selection UI & inventory deduction logic
manage_tickets.php                      → Added "Inventory" button in header

Database Tables Created:
------------------------
inventory
   - Stores all replacement parts with quantities
   - Columns: id, part_name, part_number, quantity, notes, created_at, updated_at

ticket_parts
   - Junction table linking tickets to parts used
   - Columns: id, ticket_id, part_id, quantity_used, added_at, added_by
   - Automatically deducts from inventory when parts attached
